# 사단법인 더나일 홈페이지 작업 내역

## 프로젝트 개요
- **프로젝트**: 사단법인 더나일 (The NILE) 공식 홈페이지
- **기술스택**: React + Vite SPA
- **배포**: GitHub → Vercel 자동 배포
- **도메인**: www.thenile.kr
- **레포**: https://github.com/linchpin1104/thenile-homepage

## 1. 초기 세팅

### 프로젝트 구조
- Vite + React SPA 단일 파일 구조 (`thenile_website_v3.jsx`)
- `public/images/` 하위에 이미지 관리
- 폰트: Pretendard (본문), Cormorant Garamond (영문 장식), Noto Serif KR (제목)
- 컬러 테마: 네이비(`#1B2A4A`) + 골드(`#F7D76B`)

### 배포 환경
- GitHub 레포 생성 및 Vercel 연동
- 가비아 네임서버를 Vercel DNS(`ns1.vercel-dns.com`, `ns2.vercel-dns.com`)로 변경
- MX 레코드 설정 (Google Workspace 이메일 유지)
- SSL 자동 발급

## 3. 주요 기능 구현

### 히어로 배경 이미지 롤링
- 7장의 배너 이미지가 자동 전환 (5초 간격)
- 어두운 오버레이 + 페이드 트랜지션
- 하단 인디케이터 (클릭으로 이동 가능)

### Vision 카드 (클릭 플립)
- 3개의 비전 카드를 클릭하면 뒤집어지며 상세 설명 표시
- 다시 클릭하면 원래대로 복원

### 사업소개 이미지 슬라이더
- 각 사업별 최대 5장 이미지 (`public/images/projects/{폴더}/1.jpg~5.jpg`)
- 이미지 비율 자동 판별: 가로형 → cover, 세로형(포스터) → contain
- 네이비 오버레이(`rgba(27,42,74,.1)`)로 색감 통일
- 좌우 화살표 + 하단 인디케이터

### URL 라우팅
- `window.history.pushState` 기반 SPA 라우팅
- 브라우저 뒤로가기/앞으로가기 지원
- 페이지별 URL 부여

### 후원하기 외부 링크
- 후원하기 버튼 → donationbox 외부 결제 페이지 연결
- URL: `https://link.donationbox.co.kr/donationBoxJoin.jsp?campaignuid=1FuNiwn6W6`

### 협력문의 폼
- 문의 유형 선택 (칩 버튼)
- 입력 필드: 기관명, 담당자명, 이메일, 연락처, 문의 내용
- `mailto:` 방식으로 이메일 전송

## 8. 도메인 & DNS 설정
